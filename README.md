# Discord Bot for NewebPay(Cutespirit-SHOP)
It is a bot for getting value of Cutespirit-SHOP pay website by using python.

## Requirements
| Name | pip version |
| ---- | ----------- |
| discord.py | 3 up |
| selenium | 3 up |

## Installation
bot.ini
```json
{
    "TOKEN" : "// Put your token here"
}
```
## Run
- python 3.9 requires!
- pip3 requires!
cmd: ``` python3 dcbot.py ```
